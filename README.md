## 📌 Descrição

Este aplicativo funciona como um **guia turístico digital**, ajudando usuários a descobrir e explorar destinos.

A plataforma fornece:

- 📍 Pontos turísticos próximos  
- 🍽️ Restaurantes e locais para comer  
- ⏱️ Tempo estimado de deslocamento  
- 📖 Informações úteis sobre cada local  

---

## 🎯 Principais Dores

- Dificuldade em encontrar locais relevantes próximos  
- Informações dispersas entre vários apps  
- Falta de contexto sobre deslocamento  
- Tempo perdido pesquisando  

---

## 👤 Personas

- **Turista** → Quer explorar uma cidade nova  
- **Morador Local** → Quer descobrir novos lugares  
- **Administrador** → Gerencia dados da plataforma  

---

## ⚙️ Funcionalidades principais

- 🔍 Busca por locais
- 🗺️ Geração de rotas inteligentes
- 💰 Filtro por preço
- ⭐ Avaliações e comentários
- 🌍 Suporte a múltiplos idiomas
